🪟 Windows 安装指引

系统要求

支持 Windows 8、Windows 10 和 Windows 11 系统
  • 下载安装
  • 导入配置
  • 开启VPN

1. 下载安装

推荐客户端

点击下载安装包，双击安装包按步骤安装

点击下载安装包

安装完成标志

安装后桌面会出现应用图标

下载问题
  • • 尝试更换浏览器
  • • 检查网络连接
  • • 关闭杀毒软件
安装问题
  • • 右键"以管理员身份运行"
  • • 关闭Windows Defender
  • • 检查系统兼容性

2. 导入配置

点击导入配置
使用下方按钮一键导入配置
确认导入成功
导入成功后，会出现配置块，如上图所示

导入遇到问题？

  • 1. 点击复制订阅链接
  • 2. 将订阅链接粘贴到软件中

3. 开启VPN

代理开关说明

  • 系统代理勾选 = 开启VPN
  • 不勾选 = 关闭VPN
找到系统托盘图标
在电脑任务栏右下角，鼠标右键小猫咪图标
开启系统代理
在右键菜单中勾选"系统代理"选项
测试连接
开启代理后，能正常访问 Google 就说明成功了
✓

连接成功标志

  • ✅ 托盘图标变色
  • ✅ 系统代理已勾选
  • ✅ 可访问Google
